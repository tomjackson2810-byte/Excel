# Excel
A collection of Excel Projects completed as part of my self-directed data analysis learning. Working with real datasets sourced from Kaggle, these projects demonstrate:
- Data cleaning and preparation, including identifying and resolving text-formatted number issues in large datasets
- Pivot Tables to summarise and analyse data across multiple dimensions
- Interactive dashboards using slicers and pivot charts
- Logical formulas including IF, nested IF, and SUMIF for data categorisation and aggregation
- XLOOKUP for targeted data retrieval across large datasets
- Conditional formatting for visual performance indicators
